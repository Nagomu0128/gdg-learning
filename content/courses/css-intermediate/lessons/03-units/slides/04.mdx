# px・rem・% の使い分け

3つの単位は、次のように使い分けるのが目安です。

- `px`: 枠線の太さや角丸など、いつでも同じ大きさにしたいもの
- `rem`: 文字サイズ。読む人のブラウザ設定に合わせてのび縮みしてほしいもの
- `%`: 「親の半分」「親いっぱい」のように、割合で決めたい幅

```css
.card {
  width: 50%; /* 親の幅の半分 */
  border-radius: 8px; /* いつでも 8px */
}

.lead {
  font-size: 1.5rem; /* ルートの 16px × 1.5 = 24px */
}
```

<Callout type="info">迷ったら「文字は rem、割合で決めたい幅は %、それ以外は px」から始めると整理しやすいです。</Callout>
