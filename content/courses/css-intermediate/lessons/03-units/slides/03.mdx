# %: 親要素のサイズが基準

`%` は「親要素のサイズを 100 とする」単位です。`width: 50%;` なら、親の幅のちょうど半分になります。

```css
.container {
  width: 800px;
}

.card {
  width: 50%;
}
```

この例では、`.card` の幅は 800px × 0.5 = **400px** です。

- 基準はいつも「親」。親の幅が変われば、% で書いた幅も自動で変わります
- 何を 100 とするかはプロパティごとに決まっています(width の % は親の幅、font-size の % は親の文字サイズ)

<Callout type="warn">親のサイズがはっきり決まっていないと、% は思いどおりに働かないことがあります。まず親のサイズを確かめましょう。</Callout>
