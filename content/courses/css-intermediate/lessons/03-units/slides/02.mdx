# rem: ルートの文字サイズが基準

`rem` は「ルート(html 要素)の文字サイズを 1 とする」単位です。ブラウザの標準では、ルートの文字サイズは **16px** です。

```css
.lead {
  font-size: 1.5rem;
}
```

- `1rem` = 16px × 1 = 16px
- `1.5rem` = 16px × 1.5 = 24px
- `2rem` = 16px × 2 = 32px

<Callout type="info">ブラウザの設定で文字を大きくしている人には、rem で書いたサイズも連動して大きくなります。読む人の設定を尊重できる、文字サイズ向きの単位です。</Callout>
