# flex-wrap と order

flex の子は、初期設定では1行にぎゅうぎゅうに詰めこまれます。折り返しと、ならび順の変更も指定できます。

```css
.topics {
  flex-wrap: wrap;
}

.topics .pickup {
  order: -1;
}
```

- `flex-wrap: wrap;` は**親**に書きます。入りきらない子が次の行へ折り返します(初期値は nowrap)
- `order` は**子**に書きます。初期値は 0 で、数の小さい順に前からならぶので、`-1` にすると先頭に出ます

<Callout type="warn">order が変えるのは見た目のならびだけです。HTML の順番は変わらないので、読み上げソフトなどでは元の順で読まれます。</Callout>
