# flex-grow: あまった幅を引き受ける

コンテナに幅があまっているとき、どの子がどれだけ広がるかは、子に書く `flex-grow` で決めます。

```css
.main {
  flex-grow: 1;
}
```

- 初期値は `0`(あまりが出ても広がらない)
- 1つの子だけ `flex-grow: 1;` にすると、その子があまりを全部引き受けます
- 2つの子に `1` と `1` を書けば半分ずつ、`2` と `1` なら 2 対 1 で分け合います

<Callout type="info">広がるのは「あまった幅」の分だけです。スタート時点の幅そのものは、次のスライドの flex-basis で決めます。</Callout>
