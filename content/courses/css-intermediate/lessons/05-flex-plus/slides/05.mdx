# やってみよう

ニュースサイトのレイアウトを仕上げます。`style.css` を編集しましょう。

1. `.main` に `flex-grow: 1;` を書いて、あまった幅をメイン記事に
2. `.side` に `flex-basis: 200px;` を書いて、サイドメニューの基準幅を決める
3. `.topics` に `flex-wrap: wrap;` を書いて、トピックを折り返す
4. `.topics .pickup` に `order: -1;` を書いて、「特集」を先頭に動かす

プレビューで、メイン記事が広がり、トピックが2行にならんで「特集」が先頭に来るのを確かめましょう。

書けたら「できた!」ボタンで判定します。
