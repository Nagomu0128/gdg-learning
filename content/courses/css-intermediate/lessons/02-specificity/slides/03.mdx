# 詳細度: クラスは要素より強い

セレクタの強さが違うときは、書いた順番よりも**詳細度**(セレクタの具体性)が優先されます。

```css
.note {
  color: crimson;
}

p {
  color: dimgray;
}
```

```html
<p class="note">重要なお知らせ</p>
```

`.note` のほうが上に書いてあっても、この段落は crimson になります。おおまかな強さの順は次のとおりです。

1. id セレクタ(`#title`)— いちばん強い
2. クラスセレクタ(`.note`)
3. 要素セレクタ(`p`)— いちばん弱い

「全体は要素セレクタでざっくり整えて、特別な要素だけクラスで上書きする」— これが CSS の定番の設計です。

<Callout type="warn">「指定したのに効かない」ときは、より詳細度の高いルールに負けていないかを疑ってみましょう。</Callout>
