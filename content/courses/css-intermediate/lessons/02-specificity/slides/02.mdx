# カスケード: あとに書いたほうが勝つ

CSS(Cascading Style Sheets)の「カスケード」は、複数のルールがぶつかったときの**勝敗の決め方**のことです。

セレクタの強さが同じなら、**あとに書いたルール**が勝ちます。

```css
p {
  color: blue;
}

p {
  color: green;
}
```

この場合、段落の文字は green になります。同じ `p` セレクタどうしなので、下にあるほうが採用されるからです。

<Callout type="info">「上から順に読んで、最後に決めた値で上書きする」とイメージすると覚えやすいです。</Callout>
