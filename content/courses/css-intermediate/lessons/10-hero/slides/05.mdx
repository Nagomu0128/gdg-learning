# やってみよう

「そらたび」のヒーローセクションを完成させます。`style.css` を編集しましょう。

1. `.hero` のルールを作る: `height: 400px;` / `background-color: #e0f2fe;` / `display: flex;` / `justify-content: center;` / `align-items: center;` / `position: relative;`
2. `.badge` に `position: absolute;` と `top: 16px;` / `right: 16px;` を足して右上へ
3. `.cta` に `transition-property: background-color;` と `transition-duration: 0.3s;` を足す
4. `.cta:hover` のルールを作って、`background-color: #0369a1;` に変える

プレビューで、見出しが中央、バッジが右上にならび、ボタンにマウスをのせると色がふわっと変わるのを確かめましょう。

書けたら「できた!」ボタンで判定します。
