# やってみよう

ヘッダーとフッターが横断する、2列3行のページレイアウトをつくります。`style.css` を編集しましょう。

1. `.page` に `grid-template-columns: 200px 400px;` を書いて2列にする
2. つづけて `grid-template-rows: 60px 240px 60px;` を書いて3行にする
3. `header` に `grid-column-start: 1;` と `grid-column-end: 3;` を書いて、左はしから右はしまで横断させる

フッター(footer)には同じ書き方がすでに入っています。見本として参考にしましょう。

書けたら「できた!」ボタンで判定します。
