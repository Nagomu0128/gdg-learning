# 子セレクタ: ul > li

`>` は「**直下の子**だけ」を選ぶ記号です。

```css
ul > li {
  border-left-style: solid;
}
```

これは「`ul` の直下にある `li`」だけに当たります。

似た書き方に、空白で区切る `ul li`(子孫セレクタ)があります。違いを比べてみましょう。

- `ul li` — `ul` の**中にあるすべての** `li`(孫やひ孫も含む)
- `ul > li` — `ul` の**直下の** `li` だけ

今回の HTML では、`ul` の中の `li` にさらに `ol` のリストが入っています。`ul > li` を使えば、外側のリストの項目だけを選べて、`ol` の中の `li` には当たりません。

<Callout type="info">「リストの1段目だけ線を引きたい」のように、入れ子の内側を巻き込みたくないときに > が活躍します。</Callout>
