# グループ化: h1, h2

同じスタイルを複数の相手に付けたいとき、セレクタを**コンマ**で区切ってまとめられます。

```css
h1, h2 {
  color: darkslateblue;
}
```

これは、次の2つのルールを1つに束ねた書き方です。

```css
h1 {
  color: darkslateblue;
}

h2 {
  color: darkslateblue;
}
```

同じ宣言を2回書かずにすむので、あとから色を変えるときも1か所の修正で終わります。

<Callout type="info">コンマの後ろの空白はあってもなくても動きますが、「h1, h2」のように1つ空けると読みやすくなります。</Callout>
