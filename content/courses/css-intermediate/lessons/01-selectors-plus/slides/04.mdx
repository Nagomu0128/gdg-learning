# 隣接セレクタ: h2 + p

`+` は「同じ親の中で、**直後にとなり合う**要素」を選ぶ記号です。

```css
h2 + p {
  background-color: lightyellow;
}
```

これは「`h2` のすぐ後ろに続く `p`」だけに当たります。

```html
<h2>本日のおすすめ</h2>
<p>この段落には当たる</p>
<p>この段落には当たらない</p>
```

2つ目の `p` の直前にあるのは `h2` ではなく `p` なので、スタイルは付きません。

<Callout type="info">「見出しの直後の1段落だけリード文として目立たせる」など、位置関係でスタイルを変えたいときに使います。</Callout>
