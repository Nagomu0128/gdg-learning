# data-属性とは

href や src のような決まった属性のほかに、`data-` で始まる名前なら、タグに自由な情報を持たせることができます。

```html
<li data-price="150">コーヒー</li>
<li data-color="red">りんご</li>
```

- 名前は `data-` のあとに自由に決められます(半角の小文字とハイフン)
- 値は画面には表示されません。タグが裏側で覚えているメモです

<Callout type="info">
  「見た目は同じ li だけど、それぞれ違うデータを持たせたい」というときに使います。値段・色・商品番号など、何でもメモできます。
</Callout>
