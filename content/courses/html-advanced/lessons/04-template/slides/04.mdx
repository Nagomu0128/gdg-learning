# やってみよう

くだものの配列から、同じ形の li を3個ならべましょう。

1. HTML: `id="item-template"` の `<template>` を書き、中に空の `<li></li>` を1つ入れる
2. script.js: for の中のコメントの場所に、複製を作る1行を書く

```js
const clone = template.content.cloneNode(true);
```

<Callout type="info">
  複製に文字を入れてリストに追加する2行は用意してあります。プレビューに「りんご・バナナ・みかん」の3つがならんだら提出しましょう。
</Callout>
