# やってみよう

ワイヤレスヘッドホンの商品ページを、上から順に作りましょう。

1. `h1` で商品名を書く
2. `id="price"` の `p` に `data-price="4980"` を付けて「4980円」と表示する
3. `details` の中に `summary`(スペック)と、`th`・`td` の表を書く
4. `id="buy"` の「購入する」ボタンを書く
5. `id="confirm"` の `dialog` に、確認の `p` と `id="close"` の「閉じる」ボタンを入れる
6. script.js に、ダイアログを開く処理と閉じる処理を書く

<Callout type="info">
  プレビューで、スペックが開閉できること・購入ボタンでダイアログが開いて閉じられることを確かめたら提出です。ここまでできたら HTML上級は卒業です!
</Callout>
