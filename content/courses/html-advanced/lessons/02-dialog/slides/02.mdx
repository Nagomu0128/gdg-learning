# dialogタグ

`<dialog>` は、ページの上に重ねて表示する「ダイアログ」専用のタグです。書いただけでは画面に表示されません。

```html
<dialog id="confirm">
  <p>この内容で送信しますか?</p>
  <button id="close">閉じる</button>
</dialog>
```

- 中身には p や button など、普通のタグをそのまま書けます
- 表示するには JavaScript から命令します(次のスライド)

<Callout type="info">
  確認画面や通知のように「今だけ前面に出したい」UI のためのタグです。div で自作するより簡単で、キーボード操作などの面でも優れています。
</Callout>
