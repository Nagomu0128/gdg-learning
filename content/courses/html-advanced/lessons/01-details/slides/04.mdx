# やってみよう

よくある質問のページに、クリックすると答えが開く質問コーナーを作りましょう。

1. `<details>` タグを書く
2. その中の最初に `<summary>配送について</summary>` を書く
3. summary の下に `<p>` タグで答えの文を書く(例: ご注文から3日以内にお届けします。)

```html
<details>
  <summary>見出し</summary>
  <p>中身</p>
</details>
```

<Callout type="info">
  書けたらプレビューで summary をクリックして、答えが開くか試してみましょう。動いたら提出です。
</Callout>
