# やってみよう

おとどけ先フォームを仕上げましょう。コメントの番号順に4つの作業をします。

1. 最初の `<p>` の前に `<fieldset>` を書き、そのすぐ次に `<legend>お届け先</legend>` を書く
2. お名前の input に `autocomplete="name"` を付ける
3. ゆうびん番号の input に `inputmode="numeric"` と `autocomplete="postal-code"` を付ける
4. 2つ目の `</p>` の後ろに `</fieldset>` を書いて、グループを閉じる

<Callout type="info">
  うまく囲めると、プレビューで2つの入力欄のまわりに「お届け先」というタイトル付きの枠線が表示されます。書けたら「できた!」ボタンで提出しましょう。
</Callout>
