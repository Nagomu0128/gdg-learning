# ランドマークとは

nav や main のようなタグは、スクリーンリーダー(画面を音声で読み上げるソフト)にとって「ページの案内図」になります。この目印をランドマークと呼びます。

```html
<header>ページの頭(タイトルなど)</header>
<nav>メニュー</nav>
<main>ページの本文</main>
<aside>本文のわきの補足情報</aside>
<footer>ページの下部情報</footer>
```

- 利用者は「メニューへ」「本文へ」のように、ランドマーク単位でページ内をジャンプできます
- 全部 div で作ると、この案内図には何も載りません

<Callout type="warn">
  main は「このページの本文はここです」という宣言なので、1ページに1つだけにします。2つあると、どちらが本文なのか分からなくなってしまいます。
</Callout>
