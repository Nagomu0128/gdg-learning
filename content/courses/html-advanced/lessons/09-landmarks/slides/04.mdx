# やってみよう

町の図書館のページを、ランドマークで案内できるようにしましょう。

1. `aria-labelledby="menu-title"` を付けた `<nav>` を書く。中の最初に `<h2 id="menu-title">メニュー</h2>`、その下に ul・li で `<a href="#">` のリンクを2個
2. おしらせの見出しと段落を `<main>` で囲む(1つだけ)
3. 開館時間の `<p>` を入れた `<aside>` を書く
4. `<footer>` を書いて、中に `<p>` で「町の図書館」と入れる

<Callout type="info">
  見た目はほとんど変わりませんが、ページの「構造」が良くなる回です。nav の aria-labelledby と h2 の id が同じ名前になっているか、最後に見直しましょう。
</Callout>
