# 文字コードのcharset

`<meta>` は「ページ自身の情報」を書くタグで、head の中に置きます。画面には表示されません。いちばん大事なのが文字コードの指定です。

```html
<head>
  <meta charset="utf-8">
  <title>ぼくのブログ</title>
</head>
```

- `utf-8` は世界標準の文字コードで、日本語も安全に表示できます
- この指定がないと、環境によっては文字化けすることがあります

<Callout type="warn">
  charset の meta は head の中のできるだけ最初に書きます。ブラウザが文字コードを早く知るほど安全だからです。
</Callout>
