# やってみよう

ブログの head に、3つの meta タグを書きましょう。場所は head の中、title の上です。

```html
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1">
<meta name="description" content="ページの説明文">
```

1. 文字コードの charset
2. スマホ対応の viewport
3. ページの説明の description(説明文の中身は自由に書いてかまいません)

<Callout type="info">
  meta タグは画面に出ないので、プレビューの見た目は変わりません。3行書けたらそのまま提出しましょう。
</Callout>
