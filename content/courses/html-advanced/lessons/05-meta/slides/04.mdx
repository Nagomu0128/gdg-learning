# 説明文のdescription

description は「このページは何のページか」を伝える短い説明文です。検索結果でタイトルの下に表示される、ページの自己紹介にあたります。

```html
<meta name="description" content="HTML の練習でつくった、ぼくのブログです。">
```

- 検索した人はこの文を読んで、ページを開くかどうかを決めます
- ページの内容がひと目で伝わる、短い文にします

<Callout type="info">
  meta の情報は画面には出ませんが、ブラウザのタブ・検索結果・SNS のプレビューなど、ページの「外側」で活躍します。
</Callout>
