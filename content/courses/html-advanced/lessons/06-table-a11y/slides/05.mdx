# やってみよう

文ぼうぐの値段表を、だれにでも伝わる表に仕上げましょう。コメントの番号順に4つの作業をします。

1. `<table>` の最初に `<caption>文ぼうぐの値段</caption>` を書く
2. `<thead>` の中の2つの `<th>` に `scope="col"` を付ける
3. `<tbody>` の各行の `<th>` に `scope="row"` を付ける
4. `<tbody>` の後ろに `<tfoot>` を追加して、合計の行(`<th scope="row">合計</th>` と `<td>250円</td>`)を書く

<Callout type="info">
  見た目はほとんど変わりませんが、スクリーンリーダーには大ちがいです。書けたら「できた!」ボタンで提出しましょう。
</Callout>
