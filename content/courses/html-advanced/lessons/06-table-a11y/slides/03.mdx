# th の scope — 見出しの「向き」を伝える

`<th>` は見出しセルでしたね。でも、「列(たて)の見出し」なのか「行(よこ)の見出し」なのかは、目で見ないと分かりません。`scope` 属性でその向きを伝えます。

```html
<thead>
  <tr>
    <th scope="col">商品</th>
    <th scope="col">値段</th>
  </tr>
</thead>
<tbody>
  <tr>
    <th scope="row">えんぴつ</th>
    <td>100円</td>
  </tr>
</tbody>
```

- `scope="col"` — このセルは**列**(たて方向)の見出し。column(列)の略です
- `scope="row"` — このセルは**行**(よこ方向)の見出し

<Callout type="info">
  scope があると、スクリーンリーダーは「えんぴつの値段は100円」のように、見出しと組み合わせて読み上げられます。
</Callout>
