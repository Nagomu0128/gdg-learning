# 進め方のヒント

一気に作らず、3ステップに分けるのがおすすめです。

**ステップ1: HTML の骨組み。** body の中にカードの箱を作り、3つの中身を入れます。

```html
<div class="card">
  <img src="avatar.svg" alt="アバター">
  <h1>ヤマダ タロウ</h1>
  <p>プログラミングを勉強中です。</p>
</div>
```

**ステップ2: カードの見た目。** `.card` に幅・背景色・角丸・内側の余白を付けると、一気にカードらしくなります。

**ステップ3: ホバーの仕上げ。** `.card:hover { ... }` に、マウスをのせたときの変化を1つ書きます。

プレビューで確かめながら、1ステップずつ進みましょう。
