# やってみよう

自分のプロフィールカードを作りましょう。

1. index.html に `class="card"` の箱と、`img`・`h1`・`p` の中身を書く
2. style.css で `.card` に `background-color` と `border-radius` を付ける(幅と余白もあると良いです)
3. `:hover` のルールで、マウスをのせたときの変化を付ける
4. あとは自由にアレンジして、自分らしい1枚に仕上げる

名前や紹介文は、自分のことでも架空のキャラクターでもかまいません。

できたらプレビューにマウスをのせて変化を確かめ、「できた!」ボタンで判定しましょう。
