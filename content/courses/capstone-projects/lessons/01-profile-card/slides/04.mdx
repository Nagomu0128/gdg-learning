# 自分らしくアレンジしよう

要件を満たしたら、ここからが本番です。たとえばこんなアレンジができます。

- 配色を変える: ページ背景・カード・文字の3色をそろえると、まとまって見えます
- `box-shadow` で影を付けて、カードが浮いて見えるようにする
- 趣味のタグを `ul` と `li` で足して、`display: flex` で横にならべる(CSS入門の総合レッスンと同じ形です)
- `:hover` の変化に `transition` を組み合わせて、なめらかに動かす

```css
.card {
  transition: box-shadow 0.2s;
}

.card:hover {
  box-shadow: 0 6px 16px rgba(0, 0, 0, 0.2);
}
```

<Callout type="info">transition は「変化に何秒かけるか」の指定です。ホバーの中ではなく、ふだんの .card のルールに書いておくのがコツです(のせるときも外すときも、なめらかに動きます)。</Callout>
