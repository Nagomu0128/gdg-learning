# 制作課題: プロフィールカード

応用編へようこそ。ここからは、お手本どおりに書くレッスンではなく「作品づくり」です。**要件を満たしていれば、デザインも書き方もすべて自由**です。

最初の作品は、自己紹介のプロフィールカードです。完成イメージは次のとおりです。

- ページに角丸のカードが1枚うかんでいる
- カードの中には上から、アバター画像・名前の見出し・短い紹介文がならぶ
- マウスをのせると、影が濃くなったり浮き上がったり、カードの見た目が変わる

<Callout type="info">使う道具は、これまでの HTML と CSS のレッスンで全部そろっています。エディタには index.html・style.css・script.js の3ファイルがありますが、このレッスンは JavaScript なしで完成できます。</Callout>
