# 要件 — 判定が見るもの

判定がチェックするのは次の項目**だけ**です。

index.html:

- `id="draw"` の `button` がある(おみくじを引くボタン)
- `id="result"` の要素がある(結果の表示欄。はじめの文字は自由です)

script.js の動き:

- ボタンを押すと、`#result` の文字が結果のどれかに変わる
- 何回押しても、`#result` には結果の文字列だけが入っている

結果に使える文字列は、次の一覧から**2つ以上**選んでください。判定はこの一覧と照らし合わせます。

- 大吉・中吉・小吉・吉・末吉・凶・大凶

<Callout type="warn">#result には結果の文字列をそのまま入れてください。「結果は大吉です!」のように文字を足すと判定に通りません。メッセージを添えたいときは、別の要素に表示しましょう。</Callout>
