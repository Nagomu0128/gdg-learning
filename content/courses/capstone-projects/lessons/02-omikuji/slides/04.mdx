# 自分らしくアレンジしよう

基本のおみくじができたら、自由に育てましょう。

- 候補を増やす: 一覧(大吉・中吉・小吉・吉・末吉・凶・大凶)の中から好きに追加できます
- 結果の文字を大きくする: `#result { font-size: 48px; }` などで、くじらしい迫力を出す
- 結果に合わせたひとことを、`#result` とは**別の要素**に表示する(「今日は新しいことに挑戦!」など)
- ボタンを丸く・大きくして、押したくなる見た目にする

<Callout type="info">同じ結果が続けて出るのは、ランダムでは自然なことです。気になる人は「前と同じ結果ならもう一度選び直す」処理に挑戦してみましょう。</Callout>
