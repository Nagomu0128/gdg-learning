# 制作課題: おみくじ

2つめの作品は、ボタンを押すと今日の運勢をうらなえる「おみくじ」です。

完成イメージは次のとおりです。

- 画面に「おみくじを引く」ボタンと、結果の表示欄がある
- ボタンを押すと、大吉・中吉・小吉・凶 などの結果がランダムに表示される
- 何回でも引き直せる

今回は JavaScript が主役です。「候補の配列からランダムに1つ選ぶ」という、くじ引きやゲームの定番の形を作ります。

<Callout type="info">ランダムを作る Math.random() は、このレッスンで初めて登場します。進め方のスライドで説明するので安心してください。</Callout>
