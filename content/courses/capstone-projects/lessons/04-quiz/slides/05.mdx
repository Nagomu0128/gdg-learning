# やってみよう

完成条件(= 判定が見るもの)は次の3つです。

1. script.js に questions という名前の配列で、3問ぶんの問題データがある
2. 選択肢(`class="choice"`)をクリックすると、`#question` の問題文がすぐ次の問題に切り替わる
3. 3問すべてに答えると、`#score` に半角数字入りの点数が表示される

契約さえ守れば、あとは自由です。アイデアの例:

- 「Q2 / 3」のように、いま何問目かを表示する
- 前の問題の結果(正解! / ざんねん)を、次の問題の上に小さく出す
- 点数で結果メッセージを変える(例: 全問正解なら「パーフェクト! 3問中 3問 正解!」)
- 問題を自分の好きなジャンル(音楽・ゲーム・雑学)に差し替える

<Callout type="warn">提出前に、プレビューで最初から最後まで一度プレイして、点数の表示までたしかめましょう。</Callout>
