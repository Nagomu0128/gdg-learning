# 進め方 1 — データをつくって描画する

いきなり全部を作らず、「データ → 描画 → イベント」の順で組み立てましょう。

まず問題データです。**questions という名前の配列**にします(判定は script.js にこの定義があるかを見ます)。

```js
const questions = [
  {
    text: "HTML で一番大きな見出しを作るタグはどれ?",
    choices: ["<h1>", "<p>", "<a>"],
    answer: 0, // choices の何番目が正解か
  },
  // あと2問、同じ形でつづける
];
```

次に「いま何問目か」を持つ変数と、その問題を画面に描く関数を作ります。

- `let current = 0;` — いま描くべき問題の番号
- 問題文は `id="question"` の要素の textContent へ
- 選択肢は `createElement("button")` で作り、class を `choice` にして置き場に追加

<Callout type="info">「配列のデータから画面を描く」のは、JavaScript 中級で学んだリスト描画と同じ型です。まず render() を呼んで、1問目が表示されるところまで作りましょう。</Callout>
