# TODO アプリ完全版

いよいよ応用編の最終レッスンです。学んだことをすべて使って、「やることリスト」アプリをゼロから作ります。

作るアプリでできること:

1. 入力欄にやることを書いて「追加」ボタンを押すと、リストに項目が増える
2. 項目をクリックすると「完了」になり、見た目が変わる(もう一度クリックで元に戻る)
3. 項目の削除ボタンを押すと、リストから消える
4. 「残り 2 件」のように、まだ完了していない件数がいつも表示されている

<Callout type="info">追加・完了・削除・件数表示 — この4つがそろえば、立派な「アプリ」です。コースの総仕上げとして、自分の手で組み上げましょう。</Callout>
