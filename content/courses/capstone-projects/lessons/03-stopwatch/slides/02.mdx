# 要件 — 判定が見るもの

判定がチェックするのは次の項目**だけ**です。

index.html:

- `id="display"` の要素がある(時間の表示欄。はじめの文字は `0.0`)
- `id="start"` / `id="stop"` / `id="reset"` の3つの `button` がある

script.js の動き:

- スタートを押すと時間が動き出し、**0.5秒以内**に表示が変わる(0.1秒ごとの更新がおすすめです)
- ストップを押すと表示が止まり、それ以上変わらない
- リセットを押すと表示が `0.0` に戻る

<Callout type="info">経過時間が正確かどうかは判定しません。「動く・止まる・戻る」の3つの動きができていれば合格です。表示は 1.4 のような「秒.1桁」の形にしましょう。</Callout>
