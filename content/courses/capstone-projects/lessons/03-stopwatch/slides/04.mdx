# 自分らしくアレンジしよう

動くようになったら、道具として磨きましょう。

- 二重スタートを防ぐ: スタート処理の先頭に `if (timerId !== null) { return; }` を入れる
- 動いているあいだは表示の色を変えて、「計測中」が見た目でわかるようにする
- 数字を等幅フォントにする: `#display { font-family: monospace; }` にすると、数字が増えてもガタつきません
- ボタンを横にならべて、押しやすい大きさ・色に整える

<Callout type="warn">二重スタート防止はぜひ入れましょう。setInterval を2回動かしてしまうと、タイマーが2倍の速さで進んでしまいます。</Callout>
