# やってみよう

ストップウォッチを完成させましょう。

1. index.html に `id="display"` の表示(はじめは `0.0`)と、start / stop / reset の3つの button を書く
2. script.js で「経過時間の変数」と「タイマー id の変数」を用意する
3. スタート・ストップ・リセットの3つの click 処理を書く
4. style.css で、時計らしい見た目に仕上げる

スタート → ストップ → リセットの順に押して、「動く・止まる・戻る」の3つをプレビューで確かめてから「できた!」ボタンで判定しましょう。

時間とともに動くアプリを 0 から作れたら、大きな一歩です。
