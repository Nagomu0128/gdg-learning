# 制作課題: ストップウォッチ

3つめの作品は、本物の時計のように動くストップウォッチです。

完成イメージは次のとおりです。

- 画面に「0.0」という時間の表示と、スタート・ストップ・リセットの3つのボタンがある
- スタートを押すと、0.1 → 0.2 → 0.3 と数字が増えていく
- ストップで止まり、リセットで 0.0 に戻る

「時間とともに状態が変わり、画面がそれについていく」— これはゲームやアプリの土台になる、いちばんアプリらしい形です。

<Callout type="info">主役の道具は setInterval と clearInterval(JavaScript中級のタイマーのレッスン)です。忘れていたら、先に見直しておくとスムーズです。</Callout>
