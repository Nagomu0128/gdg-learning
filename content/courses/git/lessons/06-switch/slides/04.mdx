# やってみよう

`commands.sh` で、ブランチの間を移動します。

1. `git switch feature` — feature へ移動
2. `git switch hotfix` — さらに hotfix へ移動

最後は hotfix にいる状態がゴールです。途中に `git branch` を入れると、`*` が移動していく様子を確認できます。

<Callout type="info">
「今どこにいるんだっけ?」と迷ったら `git status` の 1 行目や `git branch` の `*` を見ましょう。現在地はいつでも確認できます。
</Callout>
