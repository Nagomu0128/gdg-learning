# ブランチって何?

コミットは 1 本の線でつながっています。ブランチは、その線から新しい枝を伸ばすための「名札」です。

```
main:     初期化
             \
feature:      (ここから枝分かれして作業できる)
```

- 最初は `main` という 1 本だけがあります
- ブランチを作ると、今のコミットから新しい線を伸ばせます
- どのブランチで作業しても、他のブランチには影響しません

<Callout type="info">
ブランチを作った直後は、中身は本流とまったく同じです。そこにコミットを重ねて、はじめて内容が分かれていきます。
</Callout>
