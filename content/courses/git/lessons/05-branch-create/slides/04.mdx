# やってみよう

`commands.sh` で、2 つのブランチを作ります。

1. `git branch feature` — feature を作る(main のまま)
2. `git switch -c release` — release を作って移動する

最後に `git branch` を足すと、`main` `feature` `release` の 3 つが並び、今いる `release` に `*` が付きます。

<Callout type="info">
名前は自由ですが、`feature`(機能開発)や `release`(リリース準備)のように、役割が分かる名前をつけるのが定番です。
</Callout>
