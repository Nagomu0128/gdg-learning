# 2 種類の diff

変更は「ステージの前」と「ステージの後」で見る場所が変わります。

- `git diff` — **作業ツリー**と**ステージ**のちがい(= まだ add していない変更)
- `git diff --staged` — **ステージ**と**最後のコミット**のちがい(= add したが、まだ commit していない変更)

```
作業ツリー ──[git diff]──▶ ステージ ──[git diff --staged]──▶ 最後のコミット
```

<Callout type="info">
add した瞬間、その変更は `git diff` からは消え、`git diff --staged` の方に移ります。add でステージへ「引っ越した」からです。
</Callout>
