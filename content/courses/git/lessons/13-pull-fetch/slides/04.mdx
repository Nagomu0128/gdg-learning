# やってみよう

`commands.sh` に、同期する 2 コマンドを書きます。

1. `git fetch` — リモートの更新を取ってくる(手元はまだ変わらない)
2. `git pull origin main` — 取り込んで、手元を追いつかせる

うまくいくと `Fast-forward` と表示され、`git log --oneline` で見ると、手元の main がリモートと同じ 2 コミットにそろいます。

<Callout type="info">
今回は手元に自分の変更がなかったので、きれいに追いつけました(早送り)。手元でも別の変更をしていると、取り込みでコンフリクトが起きることもあります。そのときは git-09 と同じ手順で解決します。
</Callout>
