# やってみよう

`commands.sh` に、解決までの 4 行を書きます。

- `git merge feature` の行は、赤いエラー(コンフリクト)が出ます。それで正解です
- そのあと echo で内容を直し、add してから commit すると完了します

うまくいくと、ターミナルの最後に

```
マージを完了しました
```

と表示されます。`git status` を足すと「きれいな状態」に戻っているのが分かります。

<Callout type="info">
解決の途中で「もうやめたい」と思ったら `git merge --abort` でマージ前に戻せます。安心して試しましょう。
</Callout>
