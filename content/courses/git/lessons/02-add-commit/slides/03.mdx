# 新しい記録を 1 つ増やす

```
echo "牛乳を買う" > todo.txt
git add todo.txt
git commit -m "TODOを追加"
```

1. `echo "内容" > todo.txt` — 新しいファイルをつくる
2. `git add todo.txt` — todo.txt をステージに載せる
3. `git commit -m "メッセージ"` — ステージの内容を記録する

<Callout type="warn">
コミットには必ず `-m "メッセージ"` を付けます。「何をしたか」が後から分かる短い説明を書きましょう。
</Callout>
