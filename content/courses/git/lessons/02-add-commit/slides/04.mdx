# やってみよう

`commands.sh` に、スライドの 3 コマンドを上から順に書きます。

- すでに 1 つあるコミットに、もう 1 つ記録を積み増します
- 書くたびに右のターミナルで実行結果が再生されます
- コミットのあとに `git log --oneline` を足すと、履歴が 2 行に増えたことを確認できます(任意)

うまくいくと、ターミナルの最後に

```
[main xxxxxxx] TODOを追加
```

のような表示が出て、コミットが 2 つになります。

<Callout type="info">
「コミットする変更がありません」と出たら、git add を忘れていないか見直しましょう。
</Callout>
