# やってみよう

`commands.sh` に、push のコマンドを 1 行書きます。

```
git push -u origin feature
```

うまくいくと

```
 * [new branch] feature -> feature
ブランチ 'feature' が origin/feature を追跡するようになりました
```

のように表示され、リモートに feature ブランチができます。

<Callout type="info">
このあと feature にコミットを足したら、今度は `git push`(-u なし)だけで送れます。試してみましょう(このレッスンでは必須ではありません)。
</Callout>
