# ローカルとリモート

Git には 2 つの世界があります。

- **ローカル** … あなたのパソコンの中。init・add・commit はすべてここで完結
- **リモート(origin)** … ネット上の共有置き場(GitHub など)

```
ローカル ── git push ──▶ リモート(origin)
```

- `git push` で、手元のコミットをリモートへ送ります
- リモートに送って初めて、他の人があなたの変更を見られます

<Callout type="info">
`origin` は「メインのリモート」に付けられる定番の名前です。「送り先」くらいの意味だと思っておけば大丈夫です。
</Callout>
