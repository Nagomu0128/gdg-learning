# key で「どれがどれか」を伝える

リストの各要素には `key` という目印を付けます。React が要素を見分けるために使います。

```jsx
<ul>
  {fruits.map((fruit) => (
    <li key={fruit}>{fruit}</li>
  ))}
</ul>
```

- `key` には、要素ごとに重ならない値を渡します。今回は果物の名前そのものを使います
- `key` を付けないと React が警告を出します。並びかえや追加のときに正しく動かすための情報です

<Callout type="warn">
`key` は表示されません(画面には出ません)。React が裏で使う目印です。
</Callout>
