# map で要素に変換する

`map` は配列の1要素ずつを別のものに変え、新しい配列を作ります。文字列を `<li>` に変えてみます。

```jsx
const fruits = ["りんご", "ばなな", "みかん"];

<ul>
  {fruits.map((fruit) => (
    <li>{fruit}</li>
  ))}
</ul>
```

- `fruits.map((fruit) => ...)` は、果物1つごとに `...` を作ります
- 結果は `<li>` が3つ入った配列になり、JSX はそれをそのまま並べて表示します
- `{ }` の中に map を書くのがポイントです

<Callout type="info">
データが増えても、書くコードは同じです。配列の中身が増えれば、表示される `<li>` も自動で増えます。
</Callout>
