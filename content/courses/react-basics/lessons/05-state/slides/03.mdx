# 今の値を表示する

state の「今の値」は、ふつうの変数と同じように `{ }` で表示できます。

```jsx
const [count, setCount] = React.useState(0);

// count の今の値(はじめは 0)が表示される
<p id="count">{count}</p>
```

- 初期値に `0` を渡したので、最初の表示は `0` になります
- `count` の値が変わると、React が自動で表示も作り直してくれます

<Callout type="info">
今回は初期値を表示するところまでです。ボタンで数を増やすのは、次の「クリックに反応する」で作ります。
</Callout>
