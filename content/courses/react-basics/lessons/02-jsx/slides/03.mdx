# 書き方の型

「固定の文字」と「変数」を組み合わせるときの形です。

```jsx
const name = "タロウ";

// 変数だけを表示
<p>{name}</p>

// 文字と変数を混ぜる
<p>こんにちは、{name}さん!</p>
```

- `{ }` はタグの中身にも、文字の途中にも置けます
- クオート(`"`)で囲むと、ただの文字列になってしまいます。変数を使いたいときは `{ }` で囲みます

<Callout type="warn">
`{ }` を付け忘れて `<p>name</p>` と書くと、変数の中身ではなく「name」という文字がそのまま表示されます。
</Callout>
