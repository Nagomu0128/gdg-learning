# やってみよう

`app.jsx` の `Greeting` を完成させます。変数 `name` には「タロウ」が入っています。

```jsx
// 書き換える前
return null;

// 書き換えた後の形
return <h1>こんにちは、{name}さん!</h1>;
```

手順

1. `return null;` を `return <h1>こんにちは、{name}さん!</h1>;` に書き換える
2. 「実行」ボタンでプレビューに「こんにちは、タロウさん!」が出るか確認する
3. 「できた!」を押して判定する

<Callout type="info">
`{name}` の部分がちゃんと「タロウ」に置きかわっていれば成功です。かっこはすべて半角で書きましょう。
</Callout>
