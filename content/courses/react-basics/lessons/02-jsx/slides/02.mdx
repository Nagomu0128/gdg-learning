# 中かっこで値を埋め込む

JSX の中で `{ }` を書くと、その中身は JavaScript の式として計算され、結果が表示されます。

```jsx
function Greeting() {
  const name = "タロウ";
  return <h1>こんにちは、{name}さん!</h1>;
}
```

- `{name}` の部分が変数 `name` の中身「タロウ」に置きかわります
- 固定の文字(こんにちは、〜さん!)と混ぜて書けます
- `{ }` の中には計算式も書けます。例えば `{1 + 2}` は `3` になります

<Callout type="info">
`{ }` の中に入れられるのは「値になるもの(式)」です。変数・計算・関数の呼び出しなどが入ります。
</Callout>
