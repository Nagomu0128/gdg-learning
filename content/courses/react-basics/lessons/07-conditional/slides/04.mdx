# やってみよう

ボタンは押すたびに `on` を反転させています(`setOn(!on)`)。あとは表示を `on` に連動させます。

```jsx
// いまの状態(いつも「オフ」のまま)
<p id="status">オフ</p>

// 完成の形
<p id="status">{on ? "オン" : "オフ"}</p>
```

手順

1. `<p id="status">オフ</p>` を `<p id="status">{on ? "オン" : "オフ"}</p>` に書き換える
2. 「実行」でボタンを押し、「オン」と「オフ」が切り替わるか確認する
3. 「できた!」を押して判定する

<Callout type="info">
`on` の値(true / false)が、そのまま画面の「オン / オフ」につながります。
</Callout>
