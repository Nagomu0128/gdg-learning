# 三項演算子で切り替える

`条件 ? A : B` は「条件が true なら A、false なら B」を返す式です。JSX の `{ }` の中で使えます。

```jsx
const [on, setOn] = React.useState(false);

<p>{on ? "オン" : "オフ"}</p>
```

- `on` が `true` のときは `"オン"`、`false` のときは `"オフ"` が表示されます
- `if` 文と違い、三項演算子は「値になる」ので `{ }` の中に直接書けます
- 表示だけでなく、`{on ? <Menu /> : <Login />}` のように要素そのものも切り替えられます

<Callout type="info">
「A か B のどちらかを表示したい」ときは三項演算子がぴったりです。
</Callout>
