# 同じ部品を使い回す

props のうれしさは、同じコンポーネントを違う値で何度も使えることです。

```jsx
function App() {
  return (
    <>
      <Card name="タロウ" />
      <Card name="ハナコ" />
    </>
  );
}
```

- `Card` は1つしか作っていないのに、渡す `name` を変えるだけで表示が変わります
- 表示する中身は props 次第。部品そのものは共通のまま使い回せます

<Callout type="info">
「見た目の型は同じ、中身のデータだけ違う」ものは、props で差し替えるのが React 流です。
</Callout>
