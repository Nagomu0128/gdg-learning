# やってみよう

`App` はすでに `Card` に `name="タロウ"` を渡しています。あとは `Card` が受け取って表示するだけです。

```jsx
// いまの状態(props.name を使っていない)
return <h2>ようこそ、さん!</h2>;

// 完成の形
return <h2>ようこそ、{props.name}さん!</h2>;
```

手順

1. `Card` の `<h2>` の「さん」の前に `{props.name}` を差し込む
2. 「実行」で「ようこそ、タロウさん!」と表示されるか確認する
3. 「できた!」を押して判定する

<Callout type="info">
`{props.name}` の `{ }` を忘れると、「props.name」という文字がそのまま出てしまいます。
</Callout>
