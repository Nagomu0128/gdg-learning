# 要素は1つしか返せない

コンポーネントの `return` に要素を2つ並べると、エラーになります。

```jsx
// これはエラー(要素が2つ並んでいる)
function Menu() {
  return (
    <h2>本日のメニュー</h2>
    <p>本日のおすすめはコーヒーです</p>
  );
}
```

- `return` が返せるのは「1つのまとまり」だけです
- でも見出しと本文の2つを表示したい、という場面はよくあります

<Callout type="warn">
要素をただ並べただけでは動きません。何かで1つにまとめる必要があります。
</Callout>
