# やってみよう

`app.jsx` の `Menu` には、すでに `<>` `</>` と `<h2>` が用意されています。

```jsx
// いまの状態
<>
  <h2>本日のメニュー</h2>
</>

// 完成の形
<>
  <h2>本日のメニュー</h2>
  <p>本日のおすすめはコーヒーです</p>
</>
```

手順

1. `<h2>` のすぐ下、`<>` `</>` の内側に `<p>本日のおすすめはコーヒーです</p>` を書き足す
2. 「実行」で見出しと本文の両方が表示されるか確認する
3. 「できた!」を押して判定する

<Callout type="info">
`<p>` は必ず `<>` と `</>` の間に置きます。外に出すと1つにまとまりません。
</Callout>
