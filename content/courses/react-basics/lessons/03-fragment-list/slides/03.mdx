# フラグメントでまとめる

`<> </>`(フラグメント)で囲むと、複数の要素を1つのまとまりとして返せます。

```jsx
function Menu() {
  return (
    <>
      <h2>本日のメニュー</h2>
      <p>本日のおすすめはコーヒーです</p>
    </>
  );
}
```

- `<>` で始めて `</>` で閉じます。中には要素をいくつ並べても大丈夫です
- フラグメントは画面に余計なタグを残しません。`<div>` で囲むのと違い、見た目に影響しません

<Callout type="info">
「とりあえず1つにまとめたいけれど、余計な入れ物は増やしたくない」ときにフラグメントを使います。
</Callout>
