# やってみよう

演習では、フルネームからイニシャルを返す関数 `initials` を完成させます。

```js
initials("Taro Yamada"); // "T.Y"
initials("jiro suzuki"); // "J.S"(小文字でも大文字にする)
```

手順は 3 ステップです。

1. `split(" ")` でフルネームを 2 つの単語に分ける
2. それぞれの先頭の文字を `[0]` で取り出し、`toUpperCase()` で大文字にする
3. テンプレートリテラルで、間にピリオドをはさんで組み立てて返す

```js
return `${first}.${last}`; // 手順 3 の形
```

書けたら「できた!」ボタンで提出しましょう。
