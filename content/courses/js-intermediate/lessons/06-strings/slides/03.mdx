# テンプレートリテラル

バッククォートという記号で文字列を囲むと、`${ }` の場所に変数の値を埋め込めます。これをテンプレートリテラルと呼びます。

```js
const name = "タロウ";
const age = 25;

// + でつなぐ書き方
console.log("私は" + name + "、" + age + "歳です");

// テンプレートリテラル(同じ結果)
console.log(`私は${name}、${age}歳です`);
```

つなぐ変数が増えるほど、テンプレートリテラルの読みやすさが生きてきます。

<Callout type="warn">囲む記号はシングルクォートではなく、バッククォートです。日本語キーボードでは Shift を押しながら @ キーで入力できます。</Callout>
