# やってみよう

カウンターの完成条件です。

1. HTML に `id="count"` の表示要素(中身は 0)と、`id="plus"` / `id="minus"` / `id="reset"` の 3 つの button を書く
2. 「+1」を押すたびに数が 1 増える
3. 「-1」を押すと数が 1 減る(0 より小さくなっても OK)
4. 「リセット」を押すと 0 に戻る

進め方のおすすめ: まず HTML の 4 つの部品を並べる → `let count = 0` と表示の書き換えを作る → +1 ボタンだけ動かす → 残り 2 つは同じ形で増やす。

<Callout type="warn">プレビューで 3 つのボタンをすべて押して、増える・減る・0 に戻るを確かめてから提出しましょう。</Callout>

すべて動いたら、「できた!」ボタンで提出です。JavaScript 中級はこれで修了です!
