# やってみよう

演習では、ボタンを押すたびにお知らせパネルの表示・非表示が切り替わるようにします。

1. ボタンとパネルの要素は初期コードで取得ずみです
2. `addEventListener` で `"click"` イベントの処理を登録しましょう
3. 処理の中で `panel.classList.toggle("hidden")` を呼びましょう

`.hidden { display: none; }` の CSS は index.html に用意してあります。

<Callout type="warn">プレビューでボタンを 2 回押して、「消える → また出る」の両方が動くことを確かめましょう。判定でも 2 回押して確認します。</Callout>

動きが確認できたら、「できた!」ボタンで提出しましょう。
