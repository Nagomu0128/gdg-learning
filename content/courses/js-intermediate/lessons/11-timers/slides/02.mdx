# setTimeout で予約する

`setTimeout(処理, 待ち時間)` と書くと、指定した時間が過ぎてから処理が 1 回だけ実行されます。待ち時間の単位はミリ秒(1000 ミリ秒 = 1 秒)です。

```js
console.log("先に表示される");

setTimeout(() => {
  console.log("300ミリ秒後に表示される");
}, 300);

console.log("これも先に表示される");
```

<Callout type="warn">setTimeout は「予約したら、待たずに次の行へ進む」のがポイントです。上の例では「先に表示される」「これも先に表示される」が先に出て、最後に予約した処理が実行されます。</Callout>
