# やってみよう

演習では、買い物リスト(`<ul id="list">`)に li 要素を JavaScript で追加します。

1. 1 個目の「りんご」はお手本として初期コードに書いてあります
2. 同じ 3 ステップで、2 個目の li「バナナ」を作って追加しましょう
3. 3 個目の li「ぶどう」も作って追加しましょう

<Callout type="info">変数名は item2、item3 のように 1 個ずつ変えましょう。同じ名前の const は 2 回宣言できません。</Callout>

プレビューにりんご・バナナ・ぶどうの 3 つが並んだら、「できた!」ボタンで提出しましょう。
