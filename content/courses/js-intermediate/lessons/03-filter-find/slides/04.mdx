# やってみよう

演習では 2 つの関数を完成させます。

1. `adults(people)` は、`age` が 20 以上(20 を含む)の人だけの配列を返します
2. `findUser(users, name)` は、`name` が一致する最初の人を返します

```js
adults(users); // 20 歳以上の人だけの配列
findUser(users, "ハナコ"); // { name: "ハナコ", age: 17 }
```

<Callout type="info">「以上」は >= で、「一致」は === で書きます。findUser では、探す名前が引数 name に入ってくることに注意しましょう。</Callout>

書けたら「できた!」ボタンで提出しましょう。
