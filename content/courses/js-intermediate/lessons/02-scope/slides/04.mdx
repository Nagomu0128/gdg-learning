# やってみよう

演習では、0 から 2 回カウントアップして `[1, 2]` を返す関数 `counterDemo` を完成させます。

```js
counterDemo(); // [1, 2] を返す
```

やることは 2 つです。

1. 関数の最初で、`let` を使って `count` を宣言し、`0` を入れる
2. お手本と同じ 2 行で、2 回目のカウントアップを書く

<Callout type="info">`count` は必ず関数の中で宣言しましょう。外で宣言すると、呼び出すたびに結果が変わる関数になってしまいます。</Callout>

書けたら「できた!」ボタンで提出しましょう。
