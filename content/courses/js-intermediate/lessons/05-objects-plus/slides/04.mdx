# やってみよう

演習では 2 つの関数を完成させます。

1. `introduce(person)` は、分割代入で `name` と `age` を取り出し、「タロウは25歳です」の形の文字列を返します
2. `merge(obj1, obj2)` は、スプレッド構文で 2 つをあわせた新しいオブジェクトを返します

```js
introduce({ name: "タロウ", age: 25 }); // "タロウは25歳です"
merge({ a: 1 }, { b: 2 }); // { a: 1, b: 2 }
```

<Callout type="info">merge では obj1、obj2 の順に展開します。同じキーがあったときは、あとの obj2 の値が残ります。</Callout>

書けたら「できた!」ボタンで提出しましょう。
