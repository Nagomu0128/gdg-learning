# 関数式: 関数を変数に入れる

名前を付けずに作った関数を、変数に代入できます。これを「関数式」と呼びます。

```js
// 関数宣言(いままでの書き方)
function greet1(name) {
  return "こんにちは、" + name + "さん";
}

// 関数式(関数を変数に入れる)
const greet2 = function (name) {
  return "こんにちは、" + name + "さん";
};

console.log(greet2("タロウ")); // こんにちは、タロウさん
```

呼び出し方はどちらも同じで、変数名のうしろにカッコを付けます。

<Callout type="warn">関数式は「その行が実行されてはじめて」変数に入ります。定義した行より上では呼び出せません。</Callout>
