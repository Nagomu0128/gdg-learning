# アロー関数

関数式は、`=>`(矢印)を使った「アロー関数」で、もっと短く書けます。

```js
// 関数式
const twice1 = function (n) {
  return n * 2;
};

// アロー関数(上と同じ意味)
const twice2 = (n) => n * 2;

console.log(twice2(5)); // 10
```

`=>` の左が引数、右が「返す値」です。返す式を 1 つ書くだけなら、`return` も波かっこもいりません。

<Callout type="info">処理が複数行になるときは `(n) => { ... }` と波かっこで囲み、`return` を書きます。配列メソッドで見た `numbers.map((n) => n * 2)` のカッコの中身も、このアロー関数です。</Callout>
