# やってみよう

演習では、名前を入力すると `#preview` にあいさつが表示される「あいさつメーカー」を完成させます。

1. 入力欄と表示欄の要素は初期コードで取得ずみです
2. `addEventListener` で `"input"` イベントの処理を登録しましょう
3. 処理の中で `preview.textContent` を「こんにちは、(入力した名前)さん!」に更新しましょう。名前の部分は `input.value` です

<Callout type="warn">表示する文字は「こんにちは、」と「さん!」も含めて正確に。プレビューで名前を打ちながら、表示がついてくることを確かめましょう。</Callout>

動きが確認できたら、「できた!」ボタンで提出しましょう。
