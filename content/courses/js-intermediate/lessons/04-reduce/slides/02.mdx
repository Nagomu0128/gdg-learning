# reduce の形

`reduce` は、「これまでの累計」と「今の要素」から次の累計を作る、をくり返すメソッドです。

```js
const numbers = [100, 200, 300];

const result = numbers.reduce((sum, n) => sum + n, 0);

console.log(result); // 600
```

アロー関数の 1 つ目の引数 `sum` が累計、2 つ目の `n` が今の要素です。最後の `0` は累計のスタート値(初期値)です。動きを追うとこうなります。

- 累計 0 + 100 で 100
- 累計 100 + 200 で 300
- 累計 300 + 300 で 600

<Callout type="info">最後まで処理した累計が、reduce 全体の結果として返ります。</Callout>
