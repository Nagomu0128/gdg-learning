# やってみよう

演習では 2 つの関数を完成させます。

1. `total(numbers)` は、reduce で合計を返します(初期値 0 を忘れずに)
2. `average(numbers)` は、total を使って平均を返します

```js
total([100, 200, 300]); // 600
average([10, 20, 30]); // 20
```

<Callout type="info">average の中では、total(numbers) を numbers.length で割ります。</Callout>

書けたら「できた!」ボタンで提出しましょう。
