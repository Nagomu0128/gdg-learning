# Set — 重複のない集まり

`Set` は「同じ値は1つしか入らない」データの入れ物です。`new Set(配列)` で作ると、重複が自動的に取り除かれます。

```js
const tags = new Set(["赤", "青", "赤"]);
console.log(tags.size); // 2(「赤」は1つにまとまる)

tags.add("緑"); // 追加する
console.log(tags.has("青")); // true(入っているか調べる)
```

- 個数は `.size` で取れます。`( )` を付けないプロパティです
- 「配列の中にある種類の数」は `new Set(配列).size` の1行でわかります
