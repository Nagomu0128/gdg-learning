# やってみよう

2つの関数を完成させます。

1. `uniqueCount(items)`: 配列 `items` の「重複を除いた種類の数」を返します。`new Set(items)` を作り、その `.size` を `return` しましょう
2. `scoreOf(subject)`: 教科名から点数を返します。`new Map()` を作り、`set` で「算数は 90、国語は 75、理科は 60」を登録して、`get(subject)` の結果を `return` しましょう

<Callout type="warn">2つとも function 宣言のまま書きます(判定がこの名前の関数を呼び出します)。</Callout>
