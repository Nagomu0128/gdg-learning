# やってみよう

おすすめの手順です。1つできるたびに提出して、判定のメッセージを道しるべにしましょう。

1. HTML: 仕様の4つの部品(入力欄、追加ボタン、ul、残数)を書きます。`#count` の最初の中身は 0 にします
2. JS: `todos` 配列と `render()` を書き、最後に一度 `render()` を呼びます
3. 追加: `#add` のクリックで `todos.push({ text: input.value, done: false })` してから `render()` します
4. 完了と削除: `#list` のクリック1つで仕分けて、配列を変えたら `render()` します

<Callout type="warn">入力欄が空のときは追加しない、追加したら入力欄を空に戻す、といった仕上げも自由に足してみましょう(判定には影響しません)。</Callout>
