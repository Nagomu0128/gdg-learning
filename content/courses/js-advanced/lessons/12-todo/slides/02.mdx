# 仕様 — 判定が見る id と class

次の id と class は判定に使われます。名前はこのとおりに付けましょう(それ以外の構造や見た目は自由です)。

- `id="todo-input"` : やることを入力する `<input>`
- `id="add"` : 追加ボタンの `<button>`
- `id="list"` : TODO が並ぶ `<ul>`
- `id="count"` : 残数の数字だけを入れる要素(例: `<span id="count">0</span>`)

動きの仕様は次の4つです。

1. 追加ボタンで、入力欄の文字が新しい li として増える
2. li をクリックすると class `done` が付いたり外れたりする(完了の切り替え)
3. li の中の class `delete` のボタンで、その TODO が消える
4. `#count` は「done が付いていない li の数」につねに一致する
