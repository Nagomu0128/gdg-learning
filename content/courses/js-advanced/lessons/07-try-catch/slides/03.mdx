# try...catch で受け止める

エラーが起きそうな処理を `try` のブロックで囲むと、エラーが起きた瞬間に `catch` のブロックへ飛び、プログラムは止まらずに続きます。

```js
try {
  // エラーが起きるかもしれない処理
  const data = JSON.parse(text);
  console.log("変換できました");
} catch (error) {
  // エラーが起きたときだけ、ここが実行される
  console.log("変換に失敗しました");
}
```

- エラーが起きなければ、`catch` の中は実行されません
- エラーが起きたら、`try` の残りは実行されず、すぐ `catch` に移ります
- `error` にはエラーの情報が入っています(今回は使わなくても大丈夫です)
