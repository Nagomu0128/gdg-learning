# やってみよう

フォローボタンを、state と render() の型で完成させます。

1. `render()` の中身を書きましょう。`followers` の文字は「フォロワー: (state の followers)人」、`button` の文字は following が true なら「フォロー中」、false なら「フォローする」です
2. クリックの処理を登録しましょう。following を `!` で反転し、フォローしたら followers を +1、外れたら -1 して、最後に `render()` を呼びます
3. プレビューで、押すたびに「フォロー中 ⇔ フォローする」と「101 ⇔ 100」が切り替われば完成です

<Callout type="warn">コロンは半角の `:` と半角スペース1つで「フォロワー: 100人」の形にします(判定は文字をそのまま比べます)。</Callout>
