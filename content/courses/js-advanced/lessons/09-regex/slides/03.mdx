# 数字を表す `\d` と、回数の `{n}`

- `\d` は「数字1文字(0〜9 のどれか)」を表します
- `{3}` は「直前のものを3回くり返す」です。`\d{3}` で「数字3桁」になります

```js
console.log(/\d/.test("abc")); // false(数字がない)
console.log(/\d/.test("abc5")); // true

console.log(/\d{3}/.test("12")); // false(3桁そろっていない)
console.log(/\d{3}/.test("123")); // true
```

<Callout type="info">`[0-9]` も `\d` と同じ「数字1文字」の意味です。ほかにもたくさんの記号がありますが、まずは `\d` と `{n}` から使えれば十分です。</Callout>
