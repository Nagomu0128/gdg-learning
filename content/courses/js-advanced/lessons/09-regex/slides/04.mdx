# ^ と $ で「ちょうどこの形」にする

`.test()` は「一部でも合えば true」です。「最初から最後までちょうどこの形」だけを合格にしたいときは、先頭に `^`、末尾に `$` を付けます。

```js
console.log(/\d{3}/.test("12345")); // true(途中の3桁に合ってしまう)
console.log(/^\d{3}$/.test("12345")); // false(ちょうど3桁だけが合格)
console.log(/^\d{3}$/.test("123")); // true
```

郵便番号(数字3桁、ハイフン、数字4桁)なら、こう組み立てます。

```js
const postal = /^\d{3}-\d{4}$/;
console.log(postal.test("123-4567")); // true
console.log(postal.test("1234567")); // false(ハイフンがない)
```
