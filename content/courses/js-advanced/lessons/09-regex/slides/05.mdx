# やってみよう

文字列が郵便番号の形かどうかを判定する `isPostalCode` を完成させます。

1. 「数字3桁、ハイフン、数字4桁」の正規表現を `/^ 〜 $/` の形で書きましょう
2. その正規表現の `.test(text)` の結果を `return` しましょう
3. コンソールタブで、`"123-4567"` が true、`"1234567"` が false になることを確かめましょう

<Callout type="warn">ハイフンは半角の `-` で書きます。isPostalCode は function 宣言のまま書きましょう(判定がこの名前の関数を呼び出します)。</Callout>
