# 依存配列で「いつ動くか」が決まる

`useEffect` の第2引数(依存配列)で、副作用が動くタイミングが変わります。

```jsx
React.useEffect(() => {
  // マウント時に一度だけ
}, []);

React.useEffect(() => {
  // count が変わるたびに
}, [count]);
```

- `[]` … 最初の一度だけ
- `[count]` … `count` が変わるたびに
- 依存配列を書き忘れると毎回動いてしまうので、まずは `[]` から始めましょう

<Callout type="warn">
副作用は描画が終わった「後」に走ります。だからこのレッスンの判定も、少し待ってから `#status` の中身を確認します。
</Callout>
