# 新しい配列を作って足す

配列に要素を足すときも、元を直接変えず「新しい配列」を作ります。スプレッド構文 `...` が便利です。

```jsx
setItems([...items, "バナナ"]);
```

- `[...items]` で今の要素をすべて広げた新しい配列を作ります
- そのうしろに `, "バナナ"` を足すと、末尾に1つ増えた配列になります
- これを `setItems` に渡すと、画面のリストが更新されます

<Callout type="warn">
`items.push("バナナ")` は元の配列を直接書き換えてしまい、React が変化に気づけません。必ず新しい配列を作って渡しましょう。
</Callout>
