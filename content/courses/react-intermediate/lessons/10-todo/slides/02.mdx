# 判定が見る「契約」

デザインや構造は自由ですが、判定は次の `id` / `class` を見ます。ここだけは合わせてください。

- 入力欄: `id="todo-input"`
- 追加ボタン: `id="add"`
- リスト: `id="list"` の `ul`、各タスクはその中の `li`
- 完了切替ボタン: 各 `li` の中の `class="toggle"`
- 削除ボタン: 各 `li` の中の `class="remove"`
- 完了した `li`: `className` に `"done"` を含める

<Callout type="warn">
判定はこれらの目印を頼りに動作を確認します。名前が違うと正しく判定できないので注意しましょう。
</Callout>
