# 一部だけ更新する

オブジェクトの一部を変えたいときは、元をコピーしてから上書きします。スプレッド構文 `...` を使います。

```jsx
setUser({ ...user, age: user.age + 1 });
```

- `...user` で今の `user` の中身をすべて広げます
- そのあとに `age: user.age + 1` を書くと、`age` だけ新しい値で上書きされます
- `name` はコピーされたまま残ります

<Callout type="warn">
`user.age = user.age + 1` のように元のオブジェクトを直接書き換えてはいけません。React は「新しいオブジェクトが来たか」で変化を見分けるため、必ず新しいオブジェクトを作って渡します。
</Callout>
