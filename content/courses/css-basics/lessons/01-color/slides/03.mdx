# CSSの書き方

CSSは「どこに」「なにを」「どうする」の3つで書きます。

```css
h1 {
  color: red;
}
```

- `h1` … どこに(**セレクタ**)。スタイルをつけたいタグの名前です
- `color` … なにを(**プロパティ**)。ここでは「文字の色」
- `red` … どうする(**値**)。ここでは「赤にする」

プロパティと値のあいだは `:`(コロン)で区切り、行の終わりに `;`(セミコロン)を書きます。

<Callout type="warn">記号はすべて半角で書きます。全角の「｛」や「：」はエラーのもとです。</Callout>
