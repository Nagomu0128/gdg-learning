# 背景の色とカラーコード

背景の色は `background-color` で指定します。値には `red` のような色の名前のほかに、**カラーコード**が使えます。

```css
.card {
  background-color: #f0f0f0;
}
```

カラーコードは `#` と6桁の英数字で色を表す書き方です。赤・緑・青の強さを2桁ずつ並べます。

- `#ff0000` … 赤
- `#ffffff` … 白
- `#000000` … 黒
- `#f0f0f0` … うすい灰色

<Callout type="info">色の名前にはない、こまかい色合いを表せるのがカラーコードの強みです。</Callout>
