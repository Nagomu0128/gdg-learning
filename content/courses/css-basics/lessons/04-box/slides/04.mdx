# ふちの線 border

線は「種類」と「太さ」を、それぞれのプロパティで指定します。下の線なら bottom を使います。

```css
.box {
  border-bottom-style: solid;
  border-bottom-width: 2px;
}
```

- `border-bottom-style: solid` … 下の線をふつうの1本線(実線)にする
- `border-bottom-width: 2px` … 下の線の太さを 2px にする

<Callout type="warn">線は style(種類)を指定してはじめて表示されます。width だけ書いても線は出ないので注意しましょう。</Callout>
