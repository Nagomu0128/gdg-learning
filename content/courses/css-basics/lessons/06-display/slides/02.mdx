# ブロックとインライン

HTML の要素には、大きく分けて2種類のならび方があります。

- **ブロック**: `h1` や `p`、`li` など。横いっぱいに広がり、たてに積み重なります
- **インライン**: `a` や `strong` など。文章の流れの中に横にならびます

たとえば `li` はブロックの要素なので、リストの項目はたてにならびます。

```html
<ul>
  <li>ホーム</li>
  <li>メニュー</li>
  <li>アクセス</li>
</ul>
```

<Callout type="info">このならび方は決まりきったものではなく、CSS の `display` プロパティで自由に変えられます。</Callout>
