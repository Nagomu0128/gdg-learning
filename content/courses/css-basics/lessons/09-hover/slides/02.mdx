# 擬似クラスとは

これまでのセレクタは「どの要素か」を選んでいました。擬似クラス(ぎじクラス)を使うと、「どの要素の、どんな**状態**か」まで選べます。

その代表が `:hover` です。マウスカーソルがのっているあいだだけ、スタイルが適用されます。

```css
a:hover {
  color: orange;
}
```

このルールは「`a` にマウスがのっているとき、文字色を orange にする」という意味です。カーソルが離れると、元の色に戻ります。

<Callout type="info">ほかにも、クリックした瞬間の `:active`、キーボードで選ばれたときの `:focus` などの擬似クラスがあります。</Callout>
