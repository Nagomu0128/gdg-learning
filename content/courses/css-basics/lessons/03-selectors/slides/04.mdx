# classセレクタ

`class` 属性には `.`(ドット)で始まるセレクタを使います。

```html
<li class="highlight">世界の橋の物語</li>
```

```css
.highlight {
  background-color: yellow;
}
```

`.highlight` は「class が highlight の要素」という意味です。
`class` は同じ値を何個の要素につけてもよいので、「同じ見た目にしたいなかま」をまとめて変えられます。

<Callout type="info">background-color は背景の色を変えるプロパティです。文字色の color とセットでおぼえましょう。</Callout>
