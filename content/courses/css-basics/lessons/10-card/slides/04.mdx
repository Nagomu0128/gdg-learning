# やってみよう

プロフィールカードを完成させましょう。`style.css` に自分でルールを書きます。

1. `.card` のルールを新しく作る
   - `width: 320px;`
   - `background-color: #e0f2fe;`
   - `padding-top: 24px;`(右・下・左の padding も 24px にするときれいです)
2. `.tags` のルールに足す
   - `display: flex;`
   - `column-gap: 8px;`

プレビューを見ながら、カードが形になっていく様子を楽しんでください。

これが CSS入門の最後の演習です。書けたら「できた!」ボタンで判定しましょう。
