# .refine でルールを足す

`.refine` は、スキーマに「条件の関数」を足します。関数が `true` を返す値だけが合格です。

```js
const priceSchema = z.number().refine((n) => n > 0, "正の数を入力してください");
```

- 第 1 引数が条件の関数(合格なら `true` を返す)です
- `n > 0` なので、正の数だけが合格します
- 第 2 引数は、条件を満たさなかったときのメッセージです

<Callout type="info">
`.refine` は型チェックのあとに実行されます。まず数かどうかを見て、そのあとで「正の数か」を確かめます。
</Callout>
