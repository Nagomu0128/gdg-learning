# やってみよう

`priceSchema` に `.refine` を足して、正の数だけを合格にしましょう。

```js
const priceSchema = z.number().refine((n) => n > 0, "正の数を入力してください");
```

- `100` は正の数なので OK、`-5` は正の数ではないので エラー になります
- `.refine` を足す前は、`-5` も「数」として合格してしまいます

<Callout type="info">
条件を `(n) => n >= 10` に変えると「10 以上だけ合格」になります。ルールは自由に書けます(判定は正の数で行います)。
</Callout>
