# _.keyBy で引ける表にする

`_.keyBy` は、配列を「指定したキーの値」を見出しにしたオブジェクトへ変換します。

```js
const products = [
  { id: "a1", name: "りんご" },
  { id: "b2", name: "みかん" },
];
const byId = _.keyBy(products, "id");
// → { a1: { id: "a1", name: "りんご" }, b2: { id: "b2", name: "みかん" } }
byId["a1"].name; // "りんご"
```

- 第 2 引数のキー(ここでは `"id"`)の値が、そのままオブジェクトの見出しになります
- `byId["a1"]` のように、キーから要素をすぐ取り出せます

<Callout type="info">
「一覧を id で引きたい」場面で活躍します。同じ id が複数あると、あとの要素で上書きされます。
</Callout>
