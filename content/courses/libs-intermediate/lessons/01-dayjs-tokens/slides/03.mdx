# 曜日は英語で出る

素の dayjs には日本語の設定が入っていないため、`dddd` は英語の曜日名になります。

```js
const day = dayjs("2026-01-01");
day.format("dddd"); // "Thursday"(木曜日ではない)
```

- 2026-01-01 は木曜日なので、いつ実行しても `dddd` は `Thursday` で安定します
- トークンは自由に組み合わせられます

```js
day.format("YYYY年MM月DD日 dddd"); // "2026年01月01日 Thursday"
```

<Callout type="warn">
日付は現在時刻ではなく固定の `"2026-01-01"` を使います。`dayjs()` のように引数なしで呼ぶと日によって結果が変わり、判定できません。
</Callout>
