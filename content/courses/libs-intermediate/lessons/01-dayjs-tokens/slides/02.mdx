# よく使うトークン

`format` の文字列の中で、次の大文字・小文字が日付の値に置きかわります。

```js
const day = dayjs("2026-01-01");
day.format("YYYY/MM/DD"); // "2026/01/01"
day.format("M月D日");     // "1月1日"
day.format("dddd");       // "Thursday"
```

- `MM` / `DD` はゼロ埋め、`M` / `D` はゼロ埋めなし(1 月なら `MM` は 01、`M` は 1)
- `dddd` は曜日です。トークン以外の文字(年・月・日・スラッシュなど)はそのまま出ます

<Callout type="info">
トークンは大文字と小文字で意味が変わります。まずは `YYYY` `MM` `DD` `M` `D` `dddd` を押さえれば十分です。
</Callout>
