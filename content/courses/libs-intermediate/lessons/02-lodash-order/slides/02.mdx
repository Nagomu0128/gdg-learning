# _.sortBy で 1 つのキー順に

`_.sortBy` は、指定したキーの小さい順(昇順)に並べた新しい配列を返します。

```js
const members = [
  { name: "佐藤", age: 28 },
  { name: "田中", age: 22 },
];
_.sortBy(members, "age");
// → [{ name: "田中", age: 22 }, { name: "佐藤", age: 28 }]
```

- 第 2 引数に「基準にするキー名」を文字列で渡します
- 並べ替えたあとに `.map((m) => m.name)` を続けると、名前だけの配列になります
- 元の `members` は変わりません

<Callout type="info">
数値でも文字列でも、そのキーの値を基準に小さい順で並びます。
</Callout>
