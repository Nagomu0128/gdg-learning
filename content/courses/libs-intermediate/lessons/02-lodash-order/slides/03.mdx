# _.orderBy で複数条件・向きを指定

`_.orderBy` は、キーと「昇順/降順」をそれぞれ配列で渡せます。

```js
_.orderBy(members, ["score", "age"], ["desc", "asc"]);
// score を高い順(desc)、同点なら age を若い順(asc)
```

- 第 2 引数がキーの配列、第 3 引数が向きの配列(`"asc"` = 昇順、`"desc"` = 降順)です
- 先に書いたキーが優先され、値が同じときに次のキーで並びます
- 向きを省くとすべて昇順になります

<Callout type="info">
`_.sortBy` は「昇順・1 キー」の手軽版、`_.orderBy` は「向きも複数条件も指定できる版」と覚えましょう。
</Callout>
