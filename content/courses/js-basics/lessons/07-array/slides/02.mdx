# 配列をつくる・取り出す

値をカンマで区切って `[ ]` で囲むと、配列になります。

```js
const fruits = ["りんご", "みかん", "ぶどう"];
```

配列に入っている 1 つ 1 つの値を「要素」と呼びます。要素には前から順に番号がついていて、`配列名[番号]` で取り出せます。

```js
console.log(fruits[0]); // りんご
console.log(fruits[1]); // みかん
```

<Callout type="warn">要素の番号は 1 ではなく 0 から始まります。最初の要素は `fruits[0]` です。</Callout>
