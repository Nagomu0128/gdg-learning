# やってみよう

演習では、ボタンをクリックしたら `id` が `message` の要素の文字が「クリックされました!」に変わるようにします。

1. ボタンの要素は初期コードで取得ずみです
2. `addEventListener` で `"click"` イベントの処理を登録しましょう
3. 処理の中で `message` の要素を取得し、`textContent` を書き換えましょう

<Callout type="warn">提出する前に、プレビューで実際にボタンをクリックして、文字が変わることを確かめてみましょう。</Callout>

書けたら「できた!」ボタンで提出しましょう。
