# やってみよう

`if` を上から順にならべて、`fizzbuzz` を完成させましょう。処理の流れは次のとおりです。

1. 両方の倍数なら `"FizzBuzz"` を `return`
2. 3 の倍数なら `"Fizz"` を `return`
3. 5 の倍数なら `"Buzz"` を `return`
4. どれでもなければ `String(n)` を `return`

<Callout type="info">初期コードの下にある `console.log` で、自分の関数の動きをコンソールタブから確認できます。</Callout>

書けたら「できた!」ボタンで提出しましょう。
