# 解くための道具

**倍数の判定**: `%` は割り算の余りです。余りが 0 なら割り切れる、つまり倍数です。

```js
n % 3 === 0 // n が 3 の倍数なら true
```

**「かつ」の条件**: 2 つの条件を `&&` でつなぐと「両方とも true のとき true」になります。

```js
n % 3 === 0 && n % 5 === 0 // 3 と 5 の両方の倍数
```

**数値を文字列に**: `String(n)` で数値を文字列に変えられます。

<Callout type="warn">「両方の倍数」の判定は最初に書きましょう。3 の倍数の判定を先に書くと、15 のときに `"Fizz"` が返ってしまいます。</Callout>
