# やってみよう

`main.js` の 2 つの変数を、整形した日付に書き換えましょう。

```js
const jp = day.format("YYYY年MM月DD日");
const slash = day.format("YYYY/MM/DD");
```

- `jp` は和風(2026年01月01日)、`slash` はスラッシュ区切り(2026/01/01)にします
- 「▶ 実行」でプレビューに 2 種類の日付が表示され、コンソールにも同じ文字列が出れば成功です

<Callout type="info">
書式の中身を変えるだけで表示が変わります。慣れてきたら `YYYY.MM.DD` なども試してみましょう(判定は上の 2 つで行います)。
</Callout>
