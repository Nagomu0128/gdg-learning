# format は「書式トークン」で組み立てる

`format` に渡す文字列の中の大文字部分が、日付の値に置きかわります。

```js
const day = dayjs("2026-01-01");
day.format("YYYY年MM月DD日"); // → "2026年01月01日"
day.format("YYYY/MM/DD");     // → "2026/01/01"
```

- `YYYY` = 年 4 桁、`MM` = 月 2 桁(ゼロ埋め)、`DD` = 日 2 桁(ゼロ埋め)
- 「年」「月」「日」やスラッシュなど、トークン以外の文字はそのまま出力されます
- 同じ `day` から、何種類でも好きな書式を作れます

<Callout type="info">
書式トークンは大文字・小文字で意味が変わります。まずは `YYYY` `MM` `DD` の 3 つを覚えれば十分です。
</Callout>
