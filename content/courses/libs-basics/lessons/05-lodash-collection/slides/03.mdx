# _.sumBy で合計する

`_.sumBy` は、各要素から取り出した数値を合計します。

```js
_.sumBy(items, "price");
// → 120 + 80 + 100 + 90 = 390
```

- 第 2 引数に「合計したい数値のキー名」を渡します
- 自分でループして足していく処理が 1 行になります

コンソールには、ラベルと値をカンマで並べて出せます。

```js
console.log("合計:", 390); // → 合計: 390
```

<Callout type="info">
`console.log` にカンマで複数の値を渡すと、あいだが半角スペースでつながって表示されます。
</Callout>
