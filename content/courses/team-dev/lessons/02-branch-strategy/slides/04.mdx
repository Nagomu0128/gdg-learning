# やってみよう

`commands.sh` に、feature ブランチでの作業を上から順に書きます。

- main はさわらず、まず `git switch -c` で feature ブランチへ移りましょう
- `app.js` を 1 行書きかえて、`git add` と `git commit` で記録します
- うまくいくと、main はコミット 1 つのまま、feature 側にあなたのコミットが増えます

<Callout type="warn">
赤い行はエラーです。コマンドのつづりや `"` の閉じ忘れを見直しましょう。main にコミットしてしまったときは、ブランチを切り替え忘れていないか確認します。
</Callout>
