# private と public

プロパティやメソッドの前に公開範囲を付けられます。

```ts
class SimpleCounter implements Counter {
  private count: number;

  constructor(start: number) {
    this.count = start;
  }

  public increment(): void {
    this.count = this.count + 1;
  }

  public current(): number {
    return this.count;
  }
}
```

- `private count` は「クラスの外から直接さわれない」プロパティです
- `public` なメソッドは、外から呼び出せます
- 外からは `increment()` や `current()` を通じてだけ `count` を操作できます
