# セルには2種類ある — th と td

行(`<tr>`)の中には、**セル**(表のマス目)をならべます。セルのタグは2種類あります。

- `<th>` — **見出し**のセル(table header)。太字で表示されます
- `<td>` — **データ**のセル(table data)

```html
<table>
  <tr>
    <th>名前</th>
    <th>値段</th>
  </tr>
  <tr>
    <td>りんご</td>
    <td>150円</td>
  </tr>
</table>
```

1行目が見出しの行、2行目からがデータの行です。

<Callout type="warn">

セルの数は、どの行も同じにそろえましょう。数がずれると表がくずれてしまいます。

</Callout>
