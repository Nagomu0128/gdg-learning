# 書き方の型

4つのタグを、`<body>` の中に上から順にならべるのが基本の型です。

```html
<body>
  <header>
    <h1>わたしのブログ</h1>
  </header>
  <nav>
    <a href="#">ホーム</a>
    <a href="#">プロフィール</a>
  </nav>
  <main>
    <p>ここが本文です。</p>
  </main>
  <footer>
    <p>© わたしのブログ</p>
  </footer>
</body>
```

<Callout type="info">

`href="#"` は「とりあえずリンクの形だけ作る」ときに使う書き方です。押しても同じページにとどまります。

</Callout>
