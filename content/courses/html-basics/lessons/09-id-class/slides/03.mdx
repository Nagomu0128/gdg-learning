# class — グループの名前

`class` 属性も開始タグの中に書きます。id とちがい、**同じ名前を何回でも**使えます。

```html
<p class="intro">わたしのすきな本をしょうかいします。</p>

<ul>
  <li class="item">ハリー・ポッター</li>
  <li class="item">ナルニア国物語</li>
  <li class="item">モモ</li>
</ul>
```

「同じ仲間」に同じ class を付けておくと、あとでまとめて見た目を変えられます。

- id = 1つだけの名札
- class = 仲間に付けるグループ名
