# aタグと「属性」

リンクは `<a>` タグで作ります。ただしタグ名だけでは「どこへ飛ぶか」が伝わりません。そこで使うのが「属性」です。

```html
<a href="https://example.com">れんしゅうサイト</a>
```

- 属性は開きタグの中に `名前="値"` の形で書きます
- `href` 属性の値が、リンクの飛び先(URL)になります
- タグで囲んだ文字が、クリックできる文字として表示されます

<Callout type="info">
  a は anchor(いかり)の略です。属性はこのあとのレッスンでも(画像の src など)たくさん出てきます。
</Callout>
