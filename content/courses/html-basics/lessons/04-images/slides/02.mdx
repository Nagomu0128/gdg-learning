# imgタグとsrc属性

画像は `<img>` タグでのせます。どの画像を表示するかは、`src` 属性にファイル名を書いて指定します。

```html
<img src="cat.svg" alt="ねこのイラスト">
```

- `src` は source(出どころ)の略。画像ファイルの名前や場所を書きます
- `<img>` には閉じタグがありません。開きタグ1つだけで完結します

<Callout type="info">
  今回の演習では、cat.svg という画像ファイルをあらかじめ用意してあります。ファイル名をそのまま src に書けば表示されます。
</Callout>
