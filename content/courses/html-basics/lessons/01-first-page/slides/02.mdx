# HTMLはタグで意味を伝える

Webページの中身は HTML という言語で書かれています。HTML では「タグ」という目印で文字を囲んで、ブラウザに「これは見出し」「これは段落」と意味を伝えます。

```html
<h1>こんにちは!</h1>
```

- `<h1>` が開きタグ、`</h1>` が閉じタグです
- 閉じタグには `/`(スラッシュ)がつきます
- タグで囲んだ範囲が、その意味を持つようになります

<Callout type="warn">
  タグの記号は必ず半角の `<` と `>` で書きます。全角の＜＞だと、ブラウザはタグだと分かってくれません。
</Callout>
