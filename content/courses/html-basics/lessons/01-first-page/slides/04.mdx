# titleとh1

`<title>` はページの題名です。`<head>` の中に書き、ブラウザのタブに表示されます。

```html
<head>
  <title>はじめてのページ</title>
</head>
```

`<h1>` はページでいちばん大きな見出しです。`<body>` の中に書き、画面に大きく表示されます。

```html
<body>
  <h1>こんにちは!</h1>
</body>
```

<Callout type="info">
  head は「ページの情報」、body は「画面に見える中身」。書く場所のちがいを覚えましょう。
</Callout>
