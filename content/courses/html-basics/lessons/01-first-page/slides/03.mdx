# ページ全体の骨組み

HTML のページは、いつもこの形から始まります。

```html
<!doctype html>
<html>
  <head>
    <title>ページの題名</title>
  </head>
  <body>
    <h1>ページの見出し</h1>
  </body>
</html>
```

- `<!doctype html>` : 「これは HTML の文書です」という宣言。かならず1行目に書きます
- `<html>` : ページ全体を囲む入れ物です
- `<head>` : ページの情報を書く場所。画面には表示されません
- `<body>` : 画面に表示される中身を書く場所です
