# ページ内リンク

飛び先にしたい要素に `id` を付け、`<a>` の `href` に「#」+ id の値を書きます。

```html
<a href="#about">じこしょうかいへ</a>

<section id="about">
  <h2>じこしょうかい</h2>
</section>
```

- `href="#about"` の「#」は「このページの中の id へ」という意味です
- 「#」のうしろの文字と、飛び先の id の値をぴったりそろえます

<Callout type="info">
  基礎編で学んだ id がここで活躍します。id は「ページの中で1つだけの名前」でしたね。
</Callout>
