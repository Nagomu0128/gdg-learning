# やってみよう

いよいよ自力チャレンジです。用意されているのは骨格だけ。`<body>` の中のコメントの順番どおり、6つのパーツを自分の手で組み立てましょう。

1. `<header>` — h1 と、`href="#today"` の a
2. `<article id="today">` — h2 と p
3. `<figure>` — photo.svg の img(alt 付き)と figcaption
4. 入れ子リスト — ul の項目の1つに ul、その中に li を2個
5. `<aside>`
6. `<footer>`

<Callout type="info">
  手が止まったら、ヒントやこれまでのスライドを見直しましょう(区画はレッスン1、ページ内リンクは3、figure は4、入れ子リストは5)。全部できたら、HTML中級コース修了です!
</Callout>
