# やってみよう

演習では、おやつアンケートのフォームを完成させます。

1. 最初の `<label>` に `for="snack"` を付けて、select とつなぐ
2. `<select>` の中に `<option>` を2つ追加する(ポテトチップス・グミ)
3. 「いいえ」のラジオボタンを追加する — `name` は同じ `"again"`、`id` は `"again-no"`(label も忘れずに)
4. `<label>` と `<textarea id="comment">` で「ひとこと」欄を作る

<Callout type="info">

ラジオボタンの name がそろっているか、label の for と入力欄の id がペアになっているかがポイントです。

</Callout>
