# やさしい HTML とは

ウェブページは、いろいろな人が、いろいろな方法で見ています。目の不自由な人は、画面を音声で読み上げる**読み上げソフト**(スクリーンリーダー)でページを「聞いて」います。

読み上げソフトが困るのは、**ことばになっていない情報**です。

- 画像 — ことばの説明がないと、何の画像か読めません
- 「×」のような記号だけのボタン — 押すと何が起きるのか伝わりません

このレッスンでは、だれにでも伝わるページにするための3つの書き方を学びます。

- 画像には `alt`(復習)
- 入力欄には `label` の `for`(復習)
- ことばのない部品には `aria-label`(新登場)
