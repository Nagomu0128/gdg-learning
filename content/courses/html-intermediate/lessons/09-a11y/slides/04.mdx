# やってみよう

演習では、としょかんのお知らせページを「やさしい HTML」に直します。タグはすべて書いてあるので、**属性を4つ書き足す**だけです。

1. `nav` に `aria-label="メインメニュー"` を付ける
2. `img` に `alt`(何の画像かのことば)を付ける
3. 「×」のボタンに `aria-label="お知らせを閉じる"` を付ける
4. `label` に `for="keyword"` を付けて、input の id とつなぐ

<Callout type="info">
  書き足しても見た目はほとんど変わりません。でも読み上げソフトには、まったくちがう「やさしいページ」に聞こえています。
</Callout>
