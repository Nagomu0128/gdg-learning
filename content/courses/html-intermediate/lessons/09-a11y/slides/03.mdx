# aria-labelでことばの名前を付ける

「×」と書いてあるだけのボタンは、見た目では「閉じるボタン」だとわかっても、読み上げソフトにはうまく伝わりません。

そんなときは `aria-label` 属性で、**読み上げソフトのためのことばの名前**を付けます。

```html
<button aria-label="お知らせを閉じる">×</button>
```

`nav` にも付けられます。メニューが複数あるページでも、「メインメニュー」のように名前で区別できるようになります。

```html
<nav aria-label="メインメニュー">
  <a href="#">ホーム</a>
</nav>
```

<Callout type="info">
  aria-label は画面には表示されません。見た目はそのままで、読み上げソフトへの伝わり方だけが変わります。
</Callout>
