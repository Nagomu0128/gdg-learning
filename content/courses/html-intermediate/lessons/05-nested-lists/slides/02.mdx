# リストを入れ子にする

リストの項目(`<li>`)の中に、さらに `<ul>` を書くと、リストが階層になります。

```html
<ul>
  <li>着がえ</li>
  <li>おやつ
    <ul>
      <li>チョコレート</li>
      <li>クッキー</li>
    </ul>
  </li>
</ul>
```

- 内側の `<ul>` は、`</li>` の**前**(つまり li の中)に書きます
- li の外に ul を書くと、正しい入れ子になりません

<Callout type="warn">
  閉じタグの位置に注意しましょう。「おやつ」の li は、内側のリストが終わったあとに閉じます。
</Callout>
