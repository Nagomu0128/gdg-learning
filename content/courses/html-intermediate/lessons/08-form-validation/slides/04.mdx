# pattern — 自分でルールを決める

`pattern` 属性を使うと、「数字3けた + ハイフン + 数字4けた」のような**自分で決めた形**をチェックできます。

```html
<input id="zip" type="text" pattern="[0-9]{3}-[0-9]{4}">
```

この書き方は**正規表現**といいます。意味を分解すると:

- `[0-9]` — 0から9のどれか1文字
- `{3}` — 直前のものを3回くり返す
- `-` — ハイフンそのもの

つなげると「123-4567」のような郵便番号の形になります。

<Callout type="info">

正規表現は JavaScript 上級でくわしく学びます。今回は「pattern に形を書けばチェックできる」ことがわかれば十分です。

</Callout>
