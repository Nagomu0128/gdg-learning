# やってみよう

演習では、ニュースページを区画していきます。`<h1>` と `<section>` はすでに書いてあります。

1. `<section>` の中に `<article>` を書き、中に `<h2>`(記事の見出し)と `<p>`(記事の本文)を入れる
2. `<section>` の下に `<aside>` を書き、中に `<p>` でお知らせを1行書く

書く内容の例です。

```html
<article>
  <h2>公園の桜が満開になりました</h2>
  <p>駅前の公園は、桜を見に来た人でにぎわっています。</p>
</article>
```

<Callout type="info">
  記事の見出しや本文の文章は、例と同じでなくてもかまいません。
</Callout>
