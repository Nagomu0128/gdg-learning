# asideは補足のまとまり

`<aside>` には、本文の流れから少し外れた内容 — お知らせ・関連リンク・ちょっとしたメモなど — を入れます。

```html
<aside>
  <p>お知らせ: あすは雨の予報です。</p>
</aside>
```

- 本文のまとまり(section や article)の外側に置くのが基本です
- 「aside を消しても本文の意味が変わらない」というのが目安です

<Callout type="info">
  区画のタグを使っても見た目は変わりません。ブラウザ・検索エンジン・読み上げソフトがページの構造を理解しやすくなる、意味のためのタグです。
</Callout>
