# scope — 見出しの「向き」を伝える

`<th>` は見出しのセルでした。じつは、見出しには2つの向きがあります。`scope` 属性でどちらの向きかを伝えられます。

- 列(縦方向)の見出し → `scope="col"`
- 行(横方向)の見出し → `scope="row"`

```html
<tr>
  <th scope="col">商品</th>
  <th scope="col">売上</th>
</tr>
```

見た目は変わりませんが、読み上げソフトが「このデータはどの見出しのものか」を正しく伝えられるようになります。

<Callout type="info">

今回の表の「商品」「売上」は、列(縦方向)の見出しです。なので `scope="col"` を使います。

</Callout>
