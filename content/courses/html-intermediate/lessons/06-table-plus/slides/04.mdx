# colspan — セルを横につなげる

`colspan` 属性を使うと、1つのセルを**複数の列にまたがらせる**ことができます。合計の行などでよく使います。

```html
<tr>
  <td colspan="2">合計 5000円</td>
</tr>
```

`colspan="2"` は「このセルで2列分の幅を使う」という意味です。この行はセルが1つだけでも、2列の表とぴったりそろいます。

<Callout type="warn">

colspan を使う行では、セルの数の合計が表の列数とそろうように気をつけましょう。3列の表なら「`colspan="2"` のセル + ふつうのセル1つ」のように数えます。

</Callout>
