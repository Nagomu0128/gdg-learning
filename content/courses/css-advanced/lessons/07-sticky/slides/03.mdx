# top とセットで書く

sticky には「画面のどこにはり付くか」の指定が必要です。`top: 0;` なら画面の上端です。

```css
.site-header {
  position: sticky;
  top: 0;
  background-color: #1d4ed8;
}
```

- `top: 0;` — 上端との距離が 0 になったところではり付く
- `top: 16px;` のように、少しすきまを空けてはり付かせることもできます
- はり付いたとき下の文章が透けないように、背景色も付けておきます

<Callout type="warn">position: sticky だけ書いて top を忘れると、どこにもはり付かずスクロールで流れていきます。いちばん多いミスなので、必ずセットで書きましょう。</Callout>
