# 重なり順の基本ルール

position で要素を動かすと、要素どうしが重なることがあります。何も指定しないときのルールはこの2つです。

- HTML で「あとに書いた要素」ほど手前に表示される
- position を指定した要素は、指定していない要素より手前に表示される

```html
<div class="card">おすすめカード</div>
<div class="banner">セールのバナー</div>
<!-- あとに書いた banner が card の上にかぶさる -->
```

つまり何もしないと、重なり方は HTML の順番で決まってしまいます。「カードのほうを手前にしたい」ときに使うのが、次のスライドの `z-index` です。
