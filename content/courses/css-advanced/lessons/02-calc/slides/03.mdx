# 落とし穴: スペースの位置

`calc()` には1つだけ大事なルールがあります。**`+` と `-` の前後には半角スペースが必要**です。

```css
/* 正しい */
width: calc(100% - 40px);

/* 計算されない(スペースがない) */
width: calc(100%-40px);
```

スペースを詰めると、`-40px` が「マイナス40px という1つの値」と区別できなくなるため、宣言ごと無効になってしまいます。

<Callout type="warn">calc を書いたのに幅が変わらないときは、まず `-` や `+` の前後のスペースを疑いましょう。`*` と `/` はスペースなしでも動きますが、そろえて空けておくと読みやすくなります。</Callout>
