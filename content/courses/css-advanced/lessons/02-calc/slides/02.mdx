# calc() の基本

`calc()` は、値の場所に書ける「計算式」です。足し算 `+`、引き算 `-`、かけ算 `*`、割り算 `/` が使えます。

```css
.banner {
  width: calc(100% - 40px);
}
```

これは「親要素の幅いっぱい(100%)から、40px を引いた幅」という意味です。親の幅が 800px なら 760px になります。

`calc()` のいちばんの強みは、**単位のちがう値どうしを混ぜられる**ことです。% は画面によって変わる値、px は固定の値。この2つを組み合わせた「画面いっぱいから、余白のぶんだけ引く」がひと言で書けます。

<Callout type="info">割り算は数値で割ります。`calc(100% / 4)` は「親の幅の4分の1」です。</Callout>
