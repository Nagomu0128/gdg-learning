# やってみよう

写真フェアのページを calc() で仕上げます。`style.css` を編集しましょう。

1. `.banner` に `width: calc(100% - 40px);` を書く — 左右に 20px ずつの余白が残る幅になります(左の 20px は margin で確保済み)
2. `.gallery-item` に `width: calc(100% / 4);` を書く — 4枚の写真がぴったり4等分でならびます

プレビューで、バナーの右側にも余白ができること、写真がすきまなく4等分になることを確かめましょう。

書けたら「できた!」ボタンで判定します。
