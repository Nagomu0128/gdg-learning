# ページの設計図

HTML には上から順に、次のセクションがならんでいます。

- `.site-header` — ロゴとメニュー。画面の上に追従させ、背景はテーマカラーに
- `.hero` — キャッチコピー。見出しは `.hero-title`
- `.features` — 特徴カード3枚。3列の Grid に
- `.voices` / `.site-footer` — 参加者の声とフッター。かざりは自由

判定で使う値は次のとおりです。

- テーマカラー: `--main-color: #166534;` を `:root` に定義して `var()` で使う
- `.features`: 幅 `720px`・3列(`repeat(3, 1fr)`)・列間 `24px`
- `.hero-title`: ふだんは `40px`、せまい画面では `32px`
