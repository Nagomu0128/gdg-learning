# やってみよう

`style.css` のコメント 1〜6 の順に書いていきましょう。

1. `:root` に `--main-color: #166534;` を定義する
2. `.site-header` — `background-color: var(--main-color);`、`position: sticky;`、`top: 0;`(文字色や余白は自由)
3. `.hero-title` を `font-size: 40px;` にする(hero のかざりは自由)
4. `.features` — `width: 720px;`、`display: grid;`、`grid-template-columns: repeat(3, 1fr);`、`column-gap: 24px;`
5. `.feature-card` を自由にかざる(背景色や角丸など)
6. ファイルの最後に `@media (max-width: 900px) { .hero-title { font-size: 32px; } }` を書く

スクロールしてもヘッダーが残り、カードが3列にならんだら完成です。

書けたら「できた!」ボタンで判定します。
