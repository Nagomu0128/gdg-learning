# content は必須、スタイルも付けられる

`::before` / `::after` には、**`content` プロパティが必須**です。表示する文字を引用符で囲んで指定します。content がないと、擬似要素そのものが表示されません。

擬似要素はふつうの要素と同じようにスタイルも付けられます。

```css
.item::before {
  content: "★";
  color: orange;
}
```

`color` は `::before` のルールの中に書いているので、★ だけがオレンジ色になり、項目の本文の色は変わりません。

<Callout type="warn">飾りを付けたいだけのときも `content: "";`(空文字)が必要です。content の書き忘れは「::before が効かない」原因の定番です。</Callout>
