# 擬似要素とは

擬似要素(ぎじようそ)は、要素の「一部分」や「前後」を選ぶ特別なセレクタです。コロンを**2つ**重ねて書きます。

- `::before` — 要素の中身の**先頭**に、飾りを追加する
- `::after` — 要素の中身の**末尾**に、飾りを追加する

```css
.item::before {
  content: "★";
}
```

これで、`.item` の中身の先頭に ★ が表示されます。HTML には1文字も書き足していないのがポイントです。

<Callout type="info">前のレッスンで学んだ `:hover` はコロン1つの「擬似クラス」(要素の状態を選ぶ)でした。コロン2つの「擬似要素」は、要素の一部分を選びます。</Callout>
