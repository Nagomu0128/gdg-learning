# やってみよう

カフェのナビメニューを、せまい画面では縦ならびに切り替えます。`style.css` を編集しましょう。

`.menu` はすでに `display: flex;` と `flex-direction: row;` で横ならびになっています。ここに追加します。

1. ファイルの末尾に `@media (max-width: 900px) { }` のルールを書く
2. その中に `.menu { }` を書き、`flex-direction: column;` と `row-gap: 8px;` を指定する

判定画面は幅 800px なので、正しく書けるとプレビューのメニューが縦ならびに変わります。

書けたら「できた!」ボタンで判定します。
