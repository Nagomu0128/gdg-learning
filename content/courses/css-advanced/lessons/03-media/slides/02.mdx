# @media の書き方

メディアクエリは、「条件に合う画面のときだけ」中のルールを適用するしくみです。

```css
@media (max-width: 900px) {
  .menu {
    flex-direction: column;
  }
}
```

- `(max-width: 900px)` は「画面の幅が 900px **以下**のとき」という条件です
- `{ }` の中には、ふだんと同じようにセレクタとルールを書きます。ネストが1段深くなる点に注意しましょう

<Callout type="info">逆に「〜px 以上のとき」は `(min-width: 900px)` と書きます。まずは max-width から使えれば十分です。</Callout>
