# 変数を定義する: -- と :root

カスタムプロパティは、`--` で始まる名前のプロパティとして定義します。書き方はふつうの宣言と同じ「名前: 値;」です。

```css
:root {
  --main-color: #2563eb;
}
```

`:root` はページ全体(html 要素)を指す特別なセレクタです。ここに定義した変数は、**ページのどの要素からでも使えます**。

<Callout type="info">変数名は自由に付けられます。`--main-color` や `--text-size` のように、中身がわかる名前にするのがコツです。名前の大文字と小文字は区別されます。</Callout>
