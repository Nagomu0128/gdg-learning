# aspect-ratio で枠の比率を決める

`aspect-ratio` は、要素の「幅と高さの比率」を決めるプロパティです。幅が決まっていれば、高さは比率から自動で計算されます。

```css
.photo {
  width: 240px;
  aspect-ratio: 3 / 2; /* 高さは自動で 160px になる */
}
```

- 値は `幅 / 高さ` の形で書きます。`16 / 9` はテレビのようなワイド、`1 / 1` は正方形です
- `height: 160px;` と直接書くのとちがい、あとから幅を変えても比率がくずれません

<Callout type="info">画面の大きさで幅が変わるページでも、aspect-ratio なら枠の形をいつも同じに保てます。サムネイル一覧や動画の枠づくりの定番です。</Callout>
