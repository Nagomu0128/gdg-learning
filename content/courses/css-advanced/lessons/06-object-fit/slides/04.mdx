# やってみよう

旅の思い出アルバムを仕上げます。`style.css` を編集しましょう。

いまは画像の大きさがばらばらで、高さがそろっていません。

1. `.photo-main`(大きな1枚)のコメントの場所に `aspect-ratio: 16 / 9;` と `object-fit: cover;` を書いて、ワイドな枠に切り抜く
2. `.photo`(下の3枚)のコメントの場所に `aspect-ratio: 3 / 2;` と `object-fit: cover;` を書いて、同じ形にそろえる

プレビューで、ゆがみのない、高さのそろったアルバムになったら成功です。

書けたら「できた!」ボタンで判定します。
